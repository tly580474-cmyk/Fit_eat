# 轻食刻 - 健康饮食应用设计总结

## 项目概述

**应用名称**：轻食刻（Fresh & Vitality）

**定位**：一款面向健康意识用户的饮食管理应用，帮助用户记录饮食、追踪营养、获取AI定制方案，并与社区互动。

---

## 设计系统

### 品牌理念
- **Fresh & Vitality**：清新与活力
- 风格：现代极简 + 软触觉元素
- 情感：轻松健康 - 让饮食追踪成为自然生活方式

### 色彩体系（Material Design 3）
| 色彩角色 | 色值 | 用途 |
|---------|------|------|
| Primary | #006e1c (深绿) | 主要操作、进度、成功状态 |
| Primary Container | #4caf50 (亮绿) | 卡片背景、选中态 |
| Secondary | #8b5000 (橙色) | 代谢能量、运动按钮 |
| Secondary Container | #ff9800 | 高亮、活跃状态 |
| Tertiary | #556158 (薄荷) | 辅助信息 |
| Background | #f9f9f9 (奶油白) | 主背景 |
| Error | #ba1a1a | 错误状态 |

### 字体
- **主字体**：Plus Jakarta Sans（友好圆润）
- **中文**：PingFang SC / Noto Sans SC
- **层级**：
  - Headline: 30px/700 (大标题)
  - Body: 16px/400 (正文)
  - Label: 14px/500 (标签)

### 圆角规范
- 标准卡片：16px (1rem)
- 特色卡片：24px (1.5rem)
- 按钮/胶囊：8px (0.5rem) 起

### 阴影效果
- 使用 primary 色调阴影：`0px 4px 20px rgba(0,110,28,0.06)`
- 玻璃态效果：`backdrop-filter: blur(12px)`

## 页面结构

### 移动端页面（5个主要Tab）

#### 1. 首页 - `ai_2/code.html`
- **顶部**：用户头像 + 品牌Logo + 搜索/通知
- **卡路里环形图**：今日剩余/目标/已摄入/消耗
- **每日餐单**：横向滚动卡片（早餐/午餐）
- **营养宏量**：蛋白质/碳水/脂肪进度条
- **饮食记录流**：带图片的记录列表
- **FAB**：快速添加记录

#### 2. 发现 - `desktop_2/code.html`
- **分类筛选**：全部/减脂餐/运动补给/健康食谱
- **社区动态流**：用户分享的饮食打卡
- **互动功能**：点赞/评论/分享

#### 3. 记录 - `_4/code.html`
- **日期选择器**：横向滚动日期
- **摄入量分析**：柱状图展示本周数据
- **六维营养雷达**：能量/蛋白质/脂肪/纤维/维生素/矿物质
- **AI健康建议**：智能分析与提示
- **宏量营养素**：详细进度追踪
- **饮水量追踪**：10杯可视化追踪
- **推荐文章**：健康内容入口

#### 4. 社区 - `desktop_3/code.html`
- **瀑布流布局**：3列自适应
- **内容类型**：
  - 专家分享
  - 食谱分享
  - 运动打卡
  - 小贴士
- **互动**：点赞/评论/分享

#### 5. 我的 - `_2/code.html`
- **用户资料**：头像/昵称/签名
- **数据统计**：计划天数/饮食记录/成就
- **功能菜单**：收藏/饮食计划/身体数据/设置
- **勋章墙**：已解锁/未解锁成就

### 桌面端页面

#### 桌面首页 - `desktop_1/code.html`
- 响应式布局适配宽屏
- 顶部水平导航栏
- Bento Grid 统计卡片

#### 桌面记录 - `desktop_4/code.html`
- 左侧边栏：卡路里/宏量/建议
- 右侧主区：食谱推荐/时间线记录

#### 桌面社区 - `desktop_5/code.html`
- 瀑布流3列布局
- 分类筛选栏

### AI 功能页面

#### AI定制 - Step 1 - `ai_1/code.html`
- 性别选择
- 年龄滑块
- 身高/体重输入
- 体脂率（可选）

#### AI定制 - Step 2 - `ai_2/code.html`
- AI定制减脂计划展示
- 每日热量/蛋白质/碳水/脂肪配比
- 横向日历选择
- 四餐食谱建议（早餐/午餐/加餐/晚餐）

### 食物详情页 - `_1/code.html`
- 大图展示
- 营养标签（高蛋白/低GI）
- 营养数据卡片
- Tab切换：营养概览/制作方法
- 收藏/分享功能

### 数据分析页 - `desktop_5/code.html`
- 营养雷达图
- 周热量趋势图
- 宏量营养素进度
- 饮水追踪
- AI智能建议

## 技术实现

### 前端技术栈
- **HTML5** + **Tailwind CSS**（CDN）
- **Google Material Symbols** 图标
- **SVG** 数据可视化
- **Vanilla JavaScript** 交互

### 响应式策略
- **Mobile First**：默认移动端布局
- **断点**：md (768px) / lg (1024px)
- **桌面端**：水平导航 + 侧边栏布局

### 动画效果
- **微交互**：hover scale、active scale
- **过渡**：cubic-bezier 缓动
- **视差**：滚动时 header 阴影变化
- **入场**：卡片渐入动画

## 用户画像

### 目标用户
- 年龄：20-35岁
- 特征：健康意识强、追求生活品质
- 场景：减脂/增肌/日常健康管理

### 核心功能
1. **饮食记录**：拍照/手动记录每日饮食
2. **营养追踪**：自动计算卡路里和宏量营养素
3. **AI定制**：根据身体数据生成个性化方案
4. **社区互动**：分享饮食打卡、获取灵感
5. **数据分析**：多维度营养评估报告

## 设计亮点

1. **一致的设计语言**：所有页面遵循统一的色彩、字体、圆角规范
2. **丰富的数据可视化**：环形图、柱状图、雷达图、进度条
3. **AI智能化**：个性化饮食方案、智能建议
4. **社区驱动**：用户分享、专家内容、互动激励
5. **游戏化元素**：成就系统、勋章墙、打卡记录
6. **响应式设计**：移动端和桌面端均有优化体验
