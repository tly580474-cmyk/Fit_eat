# 轻食刻 - 页面清单

## 移动端页面

| 文件夹 | 页面名称 | 功能描述 |
|--------|----------|----------|
| `_1` | 食物详情 | 展示食物营养信息、制作方法，支持收藏和添加到饮食记录 |
| `_2` | 个人中心 | 用户资料、统计数据、功能菜单、勋章墙 |
| `_3` | 社区动态 | 用户饮食打卡分享，支持点赞/评论/关注 |
| `_4` | 数据记录 | 日期选择、摄入量柱状图、六维雷达图、饮水追踪、AI建议 |
| `_5` | 首页 | 卡路里环形图、每日餐单、宏量营养素、饮食记录流 |

## AI 定制页面

| 文件夹 | 页面名称 | 功能描述 |
|--------|----------|----------|
| `ai_1` | AI定制 Step1 | 收集身体数据：性别、年龄、身高、体重、体脂率 |
| `ai_2` | AI定制 Step2 | 展示AI定制减脂计划，包含四餐食谱建议 |

## 桌面端页面

| 文件夹 | 页面名称 | 功能描述 |
|--------|----------|----------|
| `desktop_1` | 个人中心 | 响应式布局，Bento Grid统计卡片，成就墙 |
| `desktop_2` | 社区 | 分类筛选 + 瀑布流布局 |
| `desktop_3` | 社区 | 专家分享、食谱、运动打卡等内容 |
| `desktop_4` | 记录 | 左侧边栏(卡路里/宏量) + 右侧主区(食谱/时间线) |
| `desktop_5` | 食物详情 | 左侧大图 + 右侧内容(营养/AI建议/食材/步骤) |
| `ai_desktop_1` | 首页 | AI专属方案、周计划日历、饮食推荐 |
| `ai_desktop_2` | 数据分析 | 营养雷达、热量趋势、宏量进度、AI建议 |

## 核心功能模块

### 1. 饮食记录
- 拍照/手动记录
- 卡路里自动计算
- 时间线展示

### 2. 营养追踪
- 宏量营养素（蛋白质/碳水/脂肪）
- 六维雷达评估
- 周/月趋势分析

### 3. AI 定制方案
- 身体数据采集
- 个性化计划生成
- 每日食谱推荐

### 4. 社区互动
- 用户动态分享
- 专家专栏
- 点赞/评论/关注

### 5. 成就系统
- 打卡天数
- 饮食记录数
- 勋章墙
