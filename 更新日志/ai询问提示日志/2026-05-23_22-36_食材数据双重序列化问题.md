# AI 询问提示日志

**日期时间**: 2026-05-23 22:36

## 用户问题
食材显示依旧无法显示，位置上是空白的。

## 问题分析
1. `parse_recipe` 函数返回的 `ingredients` 已经是 JSON 字符串
2. 更新数据库时又调用了一次 `json.dumps`，导致双重序列化
3. 前端解析时得到的是字符串而不是列表

## 涉及文件
- `轻饮食软件后端/import_howtocook.py` - 食谱导入脚本
- `instance/light_diet.db` - SQLite 数据库

## 解决方案
直接使用 `parse_recipe` 返回的 JSON 字符串，不再进行二次序列化。
