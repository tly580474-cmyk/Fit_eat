# AI 询问提示日志：社区帖子无法显示

**日期：** 2026-05-22 23:00

## 用户反馈

在初始化数据库并正常登录后，点击加载社区内容，发现页没有任何帖子显示。

## AI 分析过程

1. 检查后端社区 API 路由 `routes/community.py`：`get_posts()` 查询 `CommunityPost` 表并分页返回
2. 检查社区数据模型 `models/community.py`：模型定义完整，关联用户和点赞/评论
3. 检查前端 `pages/discover.html`：`loadPosts()` 正确调用 API 并渲染
4. 检查 `js/api.js`：`API.community.getPosts()` 请求路径正确
5. **关键发现**：检查 `init_db.py`，发现种子数据只包含用户、食物、成就，**完全没有社区帖子数据**

## 结论

属于数据缺失问题，非代码逻辑 bug。API 和前端逻辑均正常工作，但数据库中无帖子数据可返回。

## 修复

在 `init_db.py` 中补充 8 篇社区帖子种子数据。
