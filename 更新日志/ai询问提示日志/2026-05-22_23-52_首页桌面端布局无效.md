# AI 询问提示日志：首页桌面端布局无效

**日期：** 2026-05-22 23:52

## 用户反馈

电脑端首页依旧按手机版进行排版，桌面端双栏布局未生效。

## AI 分析过程

1. 检查 `home.html` 布局结构：使用 `desktop-grid-12`、`desktop-sidebar`、`desktop-main` 自定义 CSS 类
2. 检查 `common.css`：这些类在 `@media (min-width: 1024px)` 中定义了网格布局
3. 分析兼容性问题：自定义 CSS 的 `display: grid` 和 `grid-template-columns` 可能与 Tailwind CDN 的 reset 样式冲突
4. 发现移动端/桌面端内容重复：侧边栏热量卡片与移动端环形图同时存在，未做 `lg:hidden` 处理
5. 发现 MealCard 组件有 `min-w-[260px]` 固定宽度，阻止网格列收缩

## 结论

自定义 CSS 响应式类与 Tailwind CDN 存在优先级/兼容性问题。改用 Tailwind 原生响应式类（`lg:grid-cols-12`、`lg:col-span-4` 等）可确保桌面端布局生效。

## 修复

1. 将 `desktop-grid-12` → `grid grid-cols-1 lg:grid-cols-12`
2. 将 `desktop-sidebar` → `hidden lg:flex lg:col-span-4`
3. 将 `desktop-main` → `lg:col-span-8`
4. 食谱卡片桌面端改为 3 列网格，记录改为 2 列网格
5. MealCard 添加 `lg:min-w-0`
