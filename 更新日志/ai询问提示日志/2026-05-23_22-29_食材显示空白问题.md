# AI 询问提示日志

**日期时间**: 2026-05-23 22:29

## 用户问题
主要食材显示依旧无法显示，主要食材位置上是空白的。

## 问题分析
1. HTML 中有默认的静态内容（硬编码的示例数据）
2. 如果 `food.ingredients` 为空或未定义，JavaScript 不会更新 DOM
3. 需要添加调试代码来检查后端返回的数据

## 涉及文件
- `轻饮食软件前端/pages/food-detail.html` - 食物详情页面

## 解决方案
1. 添加 `console.log` 调试日志，输出后端返回的数据
2. 即使食材为空，也更新 DOM 显示 "暂无食材信息"
3. 确保无论数据如何，都会更新 DOM
