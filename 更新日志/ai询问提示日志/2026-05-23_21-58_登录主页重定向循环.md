# AI 询问提示日志

**日期时间：** 2026-05-23 21:58

## 用户原始提问

> 我重新初始化数据库，然后一直在登陆界面和主页反复横跳出现报错如下：net ::ERR_AB0RTED http://localhost:5000/api/user/profile...

## 问题描述

用户执行 `python init_db.py --force` 重新初始化数据库后，页面在 login.html 和 home.html 之间无限循环跳转，浏览器控制台报 `ERR_ABORTED` 错误，API 请求 `/api/user/profile` 和 `/api/diet/records` 无法正常完成。

## 初步分析

- 数据库重新初始化后，原来的用户 session 失效
- 但 localStorage 中仍保留旧的 token
- login.html 页面检测到 token 存在就自动跳转到 home.html
- home.html 的 API 请求因 token 无效而失败
- 失败后未能正确清理 token，导致循环
