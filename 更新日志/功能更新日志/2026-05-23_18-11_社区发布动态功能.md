# 功能更新日志：社区发布动态功能

**日期：** 2026-05-23 18:11

## 功能描述

实现社区发布动态功能，支持文字描述和图片上传，用户可以分享自己的轻食生活。

## 功能详情

### 文字输入
- 支持最多 1000 字的文字描述
- 实时字数统计显示
- 输入框自适应高度

### 图片上传
- 支持格式：PNG、JPEG、GIF、WebP
- 最大文件大小：16MB
- 上传后实时预览
- 支持删除已选图片
- 上传进度提示

### 分类选择
提供 4 种分类供用户选择：
- 日常分享（默认）
- 减脂食谱
- 运动打卡
- 专家专栏

### 弹窗设计
- 移动端：底部弹窗样式
- 桌面端：居中模态框显示
- 点击遮罩层或按 ESC 键关闭
- 遮罩层带毛玻璃效果

### 发布规则
- 必须有文字内容或图片才能发布
- 发布成功后自动刷新帖子列表
- 发布按钮状态实时更新

## API 接口

### 图片上传
```
POST /api/upload/image
Content-Type: multipart/form-data

参数：
- file: 图片文件

返回：
{
  "success": true,
  "url": "/api/uploads/{filename}",
  "filename": "xxx.jpg"
}
```

### 发布动态
```
POST /api/community/posts
Content-Type: application/json

参数：
{
  "content": "文字内容",
  "image": "图片URL（可选）",
  "category": "分类标识"
}

返回：
{
  "success": true,
  "postId": 123
}
```

## 技术实现

1. 图片上传采用先上传后关联的方式：前端先上传图片获取 URL，再提交帖子数据
2. 后端使用 UUID 生成唯一文件名，避免冲突
3. 上传目录：`轻饮食软件后端/uploads/`
4. 弹窗使用 Tailwind CSS 实现响应式布局
5. 文件输入使用隐藏 input + 按钮触发方式
6. 表单状态通过 JavaScript 实时管理
