# 功能更新日志：饮水打卡功能

**日期：** 2026-05-22 23:59

## 功能描述

实现饮水打卡功能，支持快捷选择预设饮水量和自定义输入饮水量。

## 功能详情

### 快捷选择
提供 4 种预设饮水量供用户快速选择：
- 150ml（小杯）
- 250ml（一杯）
- 350ml（大杯）
- 500ml（大瓶）

### 自定义输入
用户可输入 1-2000ml 之间的任意数值，满足个性化饮水需求。

### 弹窗设计
- 底部弹窗样式（移动端），桌面端居中显示
- 显示今日饮水进度条
- 点击遮罩层或关闭按钮关闭

### 数据持久化
- 调用 `API.diet.updateWaterIntake(amount)` 写入后端
- 记录后自动刷新饮水数据（桌面卡片 + 移动端水杯图标）
- 桌面端和移动端饮水量显示同步更新

## 修改文件

- `轻饮食软件前端/pages/record.html` — 添加饮水打卡弹窗 HTML、JavaScript 逻辑

## 技术实现

1. 弹窗使用现有 `bottom-sheet` + `modal-overlay` 样式
2. 快捷选择使用 4 列网格布局
3. 自定义输入使用 `form-input` + 单位标签
4. `recordWater(amount)` 函数：写入 API → 刷新数据 → 关闭弹窗 → 显示 Toast
5. `updateDesktopWaterCard()` 统一更新桌面卡片、移动水杯、弹窗进度条
