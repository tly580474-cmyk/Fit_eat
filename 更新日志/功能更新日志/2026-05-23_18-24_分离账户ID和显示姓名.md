# 功能更新日志：分离账户ID和显示姓名

**日期：** 2026-05-23 18:24

## 功能描述

将登录凭据（账户ID）与显示名称（姓名）分离，用户可以独立修改这两个字段，并增加修改密码功能。

## 功能详情

### 账户ID（登录凭据）
- 用于登录系统，唯一标识用户
- 可在"修改账户ID"功能中修改
- 修改后需使用新账户ID登录
- 至少3个字符，系统内唯一

### 姓名（显示名称）
- 仅用于页面显示，不影响登录
- 可在"账号资料"中修改
- 不要求唯一性

### 修改密码
- 需要输入旧密码验证身份
- 新密码至少6个字符
- 修改成功后立即生效

## 修改文件

### 后端
- `轻饮食软件后端/models/user.py` — 添加 `account_id` 字段
- `轻饮食软件后端/routes/auth.py` — 登录/注册使用 `account_id`
- `轻饮食软件后端/routes/user.py` — 添加修改账户ID和密码的 API
- `轻饮食软件后端/migrate_account_id.py` — 数据库迁移脚本

### 前端
- `轻饮食软件前端/pages/login.html` — 登录表单改为账户ID，注册表单添加账户ID和姓名
- `轻饮食软件前端/pages/profile.html` — 添加修改账户ID和密码入口
- `轻饮食软件前端/js/common.js` — 新增 AccountIdModal 和 PasswordModal
- `轻饮食软件前端/js/api.js` — 添加修改账户ID和密码的接口

## API 接口

### 登录
```
POST /api/auth/login
{
  "accountId": "账户ID",
  "password": "密码"
}
```

### 注册
```
POST /api/auth/register
{
  "accountId": "账户ID",
  "username": "姓名",
  "email": "邮箱",
  "password": "密码"
}
```

### 修改账户ID
```
PUT /api/user/account-id
{
  "accountId": "新账户ID"
}
```

### 修改密码
```
PUT /api/user/password
{
  "oldPassword": "旧密码",
  "newPassword": "新密码"
}
```

## 数据库迁移

运行迁移脚本将现有用户的 `username` 复制到 `account_id` 字段：
```bash
cd 轻饮食软件后端
python migrate_account_id.py
```
