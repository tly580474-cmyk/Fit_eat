# 头像添加功能实现

**日期：** 2026-05-22  
**时间：** 20:52

## 修改内容

### 1. `pages/profile.html`

头像区域添加 `onclick="AvatarModal.open()"` 点击事件。

### 2. `js/common.js` — 新增 AvatarModal 组件

功能模块：
- `open()` — 打开头像选择弹窗
- `close()` — 关闭弹窗
- `selectDefault(url)` — 选择默认头像并保存
- `handleFileSelect(event)` — 处理本地文件选择，预览图片
- `uploadLocal()` — 上传本地图片作为头像

默认头像：
- 使用 DiceBear API 生成6个不同风格的卡通头像
- 点击即可直接应用

本地上传：
- 支持 JPG、PNG 格式
- 文件大小限制 5MB
- 选择后显示预览，确认后保存
- 使用 FileReader 转为 base64 存储

### 3. `routes/user.py` — 更新 API

添加 `avatar` 字段支持：
```python
if 'avatar' in data:
    user.avatar = data['avatar']
```

## API 接口

- `PUT /api/user/profile` — 更新用户资料（含头像）

## 验证

1. 登录后进入"我的"页面
2. 点击头像区域
3. 弹窗显示6个默认头像
4. 点击某个默认头像 → 提示"头像更新成功"
5. 或点击"从本地上传" → 选择图片 → 预览 → 确认使用
6. 头像更新显示
