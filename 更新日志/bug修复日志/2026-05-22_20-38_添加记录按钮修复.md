# 添加记录按钮修复

**日期：** 2026-05-22  
**时间：** 20:38

## 问题

首页"饮食记录"旁边的"添加记录"按钮无法点击，没有绑定事件。

## 修复

**文件：** `pages/home.html` (第88行)

**修改内容：** 给按钮添加 `onclick="showAddRecordModal()"` 事件

```html
<!-- 修改前 -->
<button class="flex items-center gap-1 text-green-600 ...">

<!-- 修改后 -->
<button onclick="showAddRecordModal()" class="flex items-center gap-1 text-green-600 ...">
```

## 验证

点击"添加记录"按钮后弹出底部弹窗，可正常添加饮食记录。
