# 添加饮食记录功能实现

**日期：** 2026-05-22  
**时间：** 20:32

## 修改内容

### 1. `css/common.css` — 新增弹窗样式

- `.modal-overlay` — 全屏遮罩，黑色半透明，点击可关闭
- `.bottom-sheet` — 底部弹窗容器，圆角顶部，滑入动画
- `.sheet-handle` — 顶部拖拽条
- `.meal-chip` — 餐类选择标签，选中状态为绿色边框
- `.search-result-item` — 搜索结果列表项

### 2. `js/common.js` — 新增 AddRecordModal 组件

功能模块：
- `open()` — 打开弹窗，根据当前时间自动选中餐类
- `close()` — 关闭弹窗
- `selectMeal(key)` — 切换餐类
- `onSearch(keyword)` — 防抖搜索食物库
- `selectFood(food)` — 选中搜索结果自动填充表单
- `submit()` — 调用 `API.diet.addRecord()` 提交记录

餐类自动匹配规则：
- 6:00-9:59 → 早餐
- 10:00-13:59 → 午餐
- 14:00-16:59 → 加餐
- 17:00-21:59 → 晚餐
- 22:00-5:59 → 加餐

### 3. `pages/home.html` — 修改 showAddRecordModal

- 将 toast 替换为 `AddRecordModal.open()`
- 新增 `loadPageData()` 函数用于刷新页面

### 4. `pages/record.html` — 修改 showAddRecordModal

- 同上，调用弹窗并添加刷新函数

## 测试方式

1. 启动后端 `python app.py`
2. 打开首页，点击右下角 + 按钮
3. 底部弹窗弹出，餐类自动匹配当前时段
4. 搜索食物或手动输入，填写热量和蛋白质
5. 点击确认添加，提示成功并刷新页面
