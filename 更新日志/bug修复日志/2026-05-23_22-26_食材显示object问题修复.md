# Bug 修复日志

**日期时间**: 2026-05-23 22:26

## Bug 描述
食材显示为 "eco [object Object]"，无法正确显示食材名称和用量。

## 修复内容

### 修改 `轻饮食软件前端/pages/food-detail.html`

**修复前：**
```javascript
food.ingredients.map(item =>
  `<div>...<span>${item}</span></div>`
)
```

**修复后：**
```javascript
food.ingredients.map(item => {
  const name = typeof item === 'object' ? item.name : item;
  const amount = typeof item === 'object' ? item.amount : '';
  return `<div>...<span>${name}${amount ? ' (' + amount + ')' : ''}</span></div>`;
})
```

## 修复结果
食材现在正确显示为 "食材名 (用量)" 格式，如 "鸡胸肉 (150g)"。
