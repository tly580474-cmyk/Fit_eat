# 账号资料功能实现

**日期：** 2026-05-22  
**时间：** 20:45

## 修改内容

### 1. `pages/profile.html`

给"账号资料"按钮添加 `onclick="EditProfileModal.open()"` 事件。

### 2. `js/common.js` — 新增 EditProfileModal 组件

功能模块：
- `open()` — 打开弹窗，加载当前用户信息并填充表单
- `close()` — 关闭弹窗
- `submit()` — 提交修改到后端 API

可编辑字段：
- 用户名、个人简介
- 性别、年龄
- 身高、体重、体脂率
- 目标体重、每日目标热量

### 3. `models/user.py` — 更新 to_dict()

添加 `targetCalories` 字段到用户信息返回数据中。

## API 接口

- `GET /api/user/profile` — 获取用户资料
- `PUT /api/user/profile` — 更新用户资料

## 验证

1. 登录后进入"我的"页面
2. 点击"个人设置"中的"账号资料"
3. 弹窗显示当前用户信息
4. 修改信息后点击"保存修改"
5. 提示"保存成功"并刷新页面数据
