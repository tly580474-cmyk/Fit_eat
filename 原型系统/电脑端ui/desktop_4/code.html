蓝莓坚果酸奶碗

制作步骤

1

准备基底

将希腊酸奶均匀倒入碗中，并用勺子在中心处稍微压出一个凹槽。

2

水果铺设

将洗净沥干的蓝莓整齐地摆放在酸奶一侧，保留其自然的色彩对比。

3

点缀收尾

撒上压碎的坚果和奇亚籽，最后淋上一小勺蜂蜜增加风味。建议立即食用以保证口感。
