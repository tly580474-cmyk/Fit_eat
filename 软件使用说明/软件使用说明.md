## 账号系统

### 登录

1. 打开应用，进入登录页面
2. 输入 **账户ID** 和 **密码**
3. 点击「登录」按钮

> 账户ID 是登录凭据，与显示名称（姓名）是独立的，修改姓名不会影响登录。

### 注册

1. 在登录页面点击「注册」切换到注册表单
2. 填写以下信息：
   - **账户ID**：用于登录，至少3个字符，系统内唯一
   - **姓名**：显示名称，用于页面展示
   - **邮箱**：用于找回密码等
   - **密码**：至少6个字符
3. 点击「注册」按钮完成注册

### 退出登录

进入「个人中心」页面，点击底部的「退出登录」按钮。

## 首页

首页展示以下信息：

- **今日热量统计**：已摄入热量 / 目标热量
- **营养素进度**：蛋白质、碳水、脂肪的摄入进度
- **今日饮食记录**：按餐类（早餐、午餐、晚餐、加餐）分组显示
- **快捷操作**：快速添加饮食记录、查看饮水打卡

## 饮食管理

### 添加饮食记录

1. 在首页点击「+ 添加记录」或进入饮食记录页面
2. 选择餐类：早餐 / 午餐 / 晚餐 / 加餐
3. 搜索或选择食物
4. 输入份量（默认1份）
5. 确认添加

### 查看饮食记录

- 饮食记录页面按日期分组展示
- 可切换日期查看历史记录
- 每条记录显示：食物名称、餐类、热量、蛋白质

### 食物库

- 食物库页面可浏览所有可用食物
- 支持按餐类筛选（早餐、午餐、晚餐、加餐）
- 支持按难度筛选（简单、中等、较难）
- 每个食物卡片显示：名称、热量、蛋白质、碳水、脂肪

## 饮水打卡

1. 进入「饮水」页面
2. 点击「+ 饮水」按钮
3. 选择饮水量（200ml / 350ml / 500ml）
4. 确认添加

饮水页面会展示：
- 今日饮水总量
- 饮水目标进度条
- 今日饮水记录列表

## 社区动态

### 浏览动态

进入「社区」页面，可以浏览其他用户发布的动态：
- 动态内容（文字 + 图片）
- 发布者信息（头像、姓名）
- 发布时间和位置
- 点赞数和评论数

### 发布动态

1. 在社区页面点击右下角的「+」按钮
2. 填写动态内容（文字描述）
3. 可选择上传图片（点击「+」添加图片）
4. 可选择分类：日常分享 / 食谱推荐 / 健康知识 / 运动打卡
5. 可填写位置信息
6. 点击「发布」按钮

### 互动

- 点击动态下方的点赞图标进行点赞
- 点击评论图标查看和发表评论

## 个人中心

### 查看个人信息

进入「个人中心」页面，显示：
- 头像和姓名
- 账户ID
- 个人简介
- 身体数据（身高、体重、BMI等）

### 修改个人资料

1. 点击头像区域或「编辑资料」按钮
2. 在弹窗中修改：
   - **头像**：点击头像区域上传新头像
   - **姓名**：修改显示名称（不影响登录）
   - **个人简介**：编辑个人介绍
3. 点击「保存」按钮

### 修改账户ID

1. 在个人中心页面点击「修改账户ID」
2. 输入新的账户ID（至少3个字符，系统内唯一）
3. 确认修改
4. 下次登录需使用新账户ID

### 修改密码

1. 在个人中心页面点击「修改密码」
2. 输入旧密码
3. 输入新密码（至少6个字符）
4. 确认新密码
5. 点击「确认修改」

### 设置身体数据

1. 点击「身体数据」进入设置页面
2. 填写：性别、年龄、身高、体重、体脂率（可选）
3. 设置目标：目标热量、目标体重、计划天数
4. 系统会根据数据计算BMI和推荐营养素

## 成就系统

### 查看成就

进入「成就」页面，可查看所有成就徽章：
- 已解锁的成就显示彩色图标
- 未解锁的成就显示灰色图标和解锁条件

### 成就类型

| 类型 | 说明 |
|------|------|
| 连续打卡 | 连续记录饮食的天数 |
| 饮食记录 | 累计记录饮食的次数 |
| 饮水打卡 | 累计饮水打卡次数 |
| 社区互动 | 发布动态、评论、点赞等 |
| 体重目标 | 达到目标体重 |

## 管理后台

### 访问方式

在浏览器中访问：`http://localhost:5000/admin`

> 管理后台无需登录，可直接访问。

### 功能模块

管理后台提供以下数据表的管理功能：

| 模块 | 说明 | 可用操作 |
|------|------|----------|
| 用户管理 | 查看和编辑用户信息 | 增删改查、导出 |
| 食物库 | 管理食物数据 | 增删改查、导出 |
| 饮食记录 | 查看用户饮食记录 | 增删改查、导出 |
| 饮水记录 | 查看用户饮水记录 | 增删改查 |
| 社区动态 | 管理社区帖子 | 增删改查 |
| 评论管理 | 管理评论内容 | 增删改查 |
| 成就管理 | 管理成就徽章 | 增删改查 |
| 用户成就 | 查看用户解锁的成就 | 增删改查 |
| AI身体数据 | 查看AI分析的身体数据 | 增删改查、导出 |

### 管理后台操作

- **搜索**：在搜索框输入关键词，支持模糊搜索
- **筛选**：点击筛选按钮，按条件筛选数据
- **新增**：点击「新建」按钮添加新记录
- **编辑**：点击记录右侧的编辑图标
- **删除**：勾选记录后点击删除，或点击单条记录的删除图标
- **导出**：点击「导出」按钮，选择格式（CSV等）下载数据

## 附录

### 技术栈

| 组件 | 技术 |
|------|------|
| 后端框架 | Flask |
| 数据库 | SQLite (SQLAlchemy ORM) |
| 用户认证 | Flask-Login (Session) |
| 后台管理 | Flask-Admin |
| 前端 | HTML + Tailwind CSS + 原生 JavaScript |
| 图标 | Material Symbols |

### 项目结构

```
轻食刻/
├── 轻饮食软件后端/          # 后端代码
│   ├── app.py              # 应用入口
│   ├── config.py           # 配置文件
│   ├── models/             # 数据模型
│   ├── routes/             # API 路由
│   ├── admin/              # 管理后台
│   ├── uploads/            # 上传文件存储
│   └── instance/           # SQLite 数据库
├── 轻饮食软件前端/          # 前端代码
│   ├── index.html          # 首页
│   ├── pages/              # 页面
│   ├── css/                # 样式
│   └── js/                 # 脚本
└── 软件使用说明/            # 文档
```

|------|------|------|

| `/api/user/profile` | GET | 获取个人信息 |

| `/api/user/account-id` | PUT | 修改账户ID |

| `/api/diet/records` | POST | 添加饮食记录 |
| `/api/diet/water` | GET | 获取饮水记录 |
| `/api/diet/water` | POST | 添加饮水记录 |
| `/api/food/list` | GET | 获取食物列表 |
| `/api/community/posts` | GET | 获取社区动态 |
| `/api/community/posts` | POST | 发布动态 |
| `/api/upload/image` | POST | 上传图片 |
| `/api/achievement/list` | GET | 获取成就列表 |
