# 轻食刻 - 启动步骤

## 环境要求

- Python 3.8 或更高版本
- pip（Python 包管理器，通常随 Python 一起安装）

---

## 第一步：安装依赖

打开终端（命令行），进入后端项目目录并安装所需的 Python 库：

```bash
cd D:\健康饮食软件\轻饮食软件后端
pip install -r requirements.txt
```

需要安装的依赖包：

| 包名 | 版本 | 用途 |
|------|------|------|
| Flask | 3.0.0 | Web 框架 |
| Flask-SQLAlchemy | 3.1.1 | 数据库 ORM |
| Flask-Login | 0.6.3 | 用户登录管理 |
| Flask-Admin | 1.6.1 | 后台管理界面 |
| Flask-CORS | 4.0.0 | 跨域请求支持 |
| Werkzeug | 3.0.1 | WSGI 工具库 |

---

## 第二步：初始化数据库（首次运行）

首次启动前，需要创建数据库并填充示例数据（确保当前在后端目录下）：

```bash
cd D:\健康饮食软件\轻饮食软件后端
python init_db.py
```

执行成功后会输出：

```
数据库表已创建
示例数据已填充
管理员账号: admin / admin123
测试账号: 林小悦 / 123456
```

此命令会创建 `instance/light_diet.db` 数据库文件，并填充以下初始数据：

- 2 个用户账号（管理员 + 测试用户）
- 6 个示例食物（覆盖早餐、午餐、晚餐、加餐）
- 8 个成就徽章

---

## 第三步：启动服务

确保当前在后端目录下：

```bash
cd D:\健康饮食软件\轻饮食软件后端
python app.py
```

看到以下输出即表示启动成功：

```
 * Running on all addresses (0.0.0.0)
 * Running on http://127.0.0.1:5000
```

---

## 访问地址

| 地址 | 说明 |
|------|------|
| http://localhost:5000 | 前端页面（主页） |
| http://localhost:5000/admin | 管理员后台 |

---

|------|--------|------|

## 管理员后台

访问 http://localhost:5000/admin 可直接操作数据库，支持以下数据表的增删改查：

- 用户（User）
- 食物（Food）
- 饮食记录（DietRecord）
- 饮水记录（WaterRecord）
- 社区帖子（CommunityPost）
- 评论（Comment）
- 点赞（Like）
- 关注（Follow）
- 成就（Achievement）

后台支持搜索、筛选和 CSV 导出。

---

## 停止服务

在终端中按 `Ctrl + C` 即可停止服务。

---

## 重新初始化数据库

如需清空所有数据并重新开始，进入后端目录后运行：

```bash
cd D:\健康饮食软件\轻饮食软件后端
python init_db.py
```

> **注意**：此操作会删除所有用户数据和记录，不可恢复。

---

## 常见问题

### 端口被占用

如果提示 5000 端口已被占用，可以修改 `app.py` 最后一行的端口号：

```python
app.run(debug=True, host='0.0.0.0', port=5001)  # 改为 5001
```

### 依赖安装失败

尝试使用国内镜像源：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 数据库文件位置

SQLite 数据库文件位于：

```
D:\健康饮食软件\轻饮食软件后端\instance\light_diet.db
```

删除该文件后重新运行 `init_db.py` 可重建数据库。
